# My Github Home Page

## Set up on new enviorment

```shell
npm install hexo-cli -g
cd this/repo
npm install
```

## Local test

```shell
hexo generate
hexo server
# open localhost:4000 on browser
```

## Deploy

```shell
hexo clean && hexo deploy
```

or just push.
